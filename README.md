# LeetCode Biweekly Contest — разборы

Здесь лежат мои разборы задач с контестов leetcode. Для каждой задачи — условие, ход мыслей и финальное решение на Python.

---

## Структура

Каждый файл — один контест. Внутри задачи идут в порядке возрастания сложности, как на контесте.

Для каждой задачи:
- условие с примерами
- как я к этому пришёл (не просто код, а почему именно так)
- итоговое решение с комментариями

---

## Стек

Python 3.10+. Никаких внешних библиотек — только стандартная библиотека (`collections`, `heapq` и иногда `typing` для совместимости).

## Заметки

Разборы пишу так, как думал во время контеста — включая тупики и моменты, где переосмыслил подход. Мне кажется, это полезнее чем просто готовый код.

Если нашёл ошибку или знаешь решение лучше — открывай issue.
